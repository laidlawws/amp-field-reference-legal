# AMP Field Reference Legal Page

This repository contains a simple static legal page for the AMP Field Reference app.

## Files

- `index.html` — Privacy Policy, Terms of Use, and support contact
- `styles.css` — styling for the page

## Before handing off for deployment

Update these placeholders in `index.html`:

- `help@ampos.io`
- `March 10, 2026`

## Deployment

This repository is intended to be deployed by someone else from GitHub.
No build step is required.
